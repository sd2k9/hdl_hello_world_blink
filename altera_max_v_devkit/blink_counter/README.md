HDL Hello World
===============

Function
--------
A classical simple Hello World HDL description, which
just blinks an LED with 0.5Hz and can be reset by a
push button.

Along with a simple test bench

Requirements
------------
- Device: Altera Max V
- Board: Max V Development Kit
- Software: Altera Quartus II Version 12.1
